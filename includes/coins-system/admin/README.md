# Admin - Panel de Administración

## Archivos

### coins-metabox.php
Metabox en productos para:
- Configurar precio en coins
- Habilitar/deshabilitar canje
- Ver historial de canjes

### coins-admin.php
Panel de administración:
- Página de configuración de coins
- Reportes de transacciones
- Gestión manual de saldos
- Exportar datos

## Capacidades

Permisos necesarios:
- `manage_woocommerce` - Para acceder al panel
- `edit_products` - Para editar metabox de productos

## Menús

Se agrega menú en:
- WooCommerce > Coins
  - Configuración
  - Transacciones
  - Reportes
